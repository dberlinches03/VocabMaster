# VocabMaster — Rama `Stats`

Esta rama añade un sistema de estadísticas para mejorar la experiencia del usuario y motivar el aprendizaje continuo.

## Funcionalidades incluidas
- Cálculo de palabras aprendidas (Room)
- Cálculo del **streak** de días consecutivos
- Registro del último acceso (DataStore)
- Pantalla visual con tarjetas de estadísticas
- Integración con `StatsViewModel`

## Objetivo
Proporcionar métricas claras y motivadoras que reflejen el progreso del usuario dentro de la aplicación.

## Archivos principales
- `StatsViewModel.kt`
- `StatsScreen.kt`
- `StatsUtils.kt`
