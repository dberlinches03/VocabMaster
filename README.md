# VocabMaster — Rama `Game`

Esta rama implementa un mini‑juego educativo donde el usuario debe asociar una palabra con su definición correcta.

## Funcionalidades incluidas
- Generación de una palabra aleatoria
- Creación de 3 definiciones incorrectas + 1 correcta
- Comprobación de acierto o fallo
- Colores de feedback (verde/rojo)
- Botón para pasar a la siguiente ronda
- Arquitectura MVVM con `GameViewModel`

## Objetivo
Añadir una mecánica lúdica que refuerce el aprendizaje del vocabulario mediante práctica activa.

## Archivos principales
- `GameViewModel.kt`
- `GameScreen.kt`
